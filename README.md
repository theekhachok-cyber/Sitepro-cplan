# SitePro — C Plan Design System
Construction Management Platform

## Deploy Steps (Vercel — Free)

### Step 1: GitHub
1. ไปที่ https://github.com → Sign up (ถ้ายังไม่มี)
2. คลิก "New repository"
3. ตั้งชื่อ: sitepro-cplan
4. เลือก Public → Create

### Step 2: Upload Files
อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้:
- index.html
- vercel.json

### Step 3: Vercel
1. ไปที่ https://vercel.com → Sign up ด้วย GitHub
2. New Project → Import "sitepro-cplan"
3. Framework Preset: Other
4. Deploy → รอ 1-2 นาที

### Step 4: ได้ URL
URL ที่ได้เช่น: https://sitepro-cplan.vercel.app
แชร์ให้ทีมได้เลย ใช้ได้ทุก Device

## Modules
- Dashboard (5 Roles)
- Phase 1-15 + QC
- Daily Report + GPS
- Defect & Snag List
- Procurement PR/PO
- Stock Management
- AI Document Analyzer
- Owner Portal + Milestone Approval

## Tech
- Pure HTML/CSS/JS — ไม่ต้องติดตั้งอะไร
- IBM Plex Sans Thai Font
- C Plan CI Design System
- Responsive: Desktop / Tablet / Mobile
